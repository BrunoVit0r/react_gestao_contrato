# ContractFlow Marketing — React

Estrutura convertida a partir do arquivo estático original para um pequeno projeto React (Vite).

Arquivos principais:

- [public/index.html](public/index.html)
- [src/main.jsx](src/main.jsx)
- [src/App.jsx](src/App.jsx)
- [src/index.css](src/index.css)
- [src/data.js](src/data.js)
- [src/components](src/components)

Como rodar:

1. Instale dependências:

```bash
npm install
```

2. Rode em desenvolvimento:

```bash
npm run dev
```

Observações:

- O CSS original foi copiado para `src/index.css` (algumas partes resumidas no patch; posso completar se desejar).
- Recursos externos (Bootstrap, Bootstrap Icons e AOS) estão incluídos via CDN em [public/index.html](public/index.html).
